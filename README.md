# DB Managing Console
> Local LLM + NL2SQL


## 기능

1. PDF 파싱 → 확인 및 수정 → Table 생성 및 적재
2. 자연어 → 쿼리 생성 → 확인 및 수정 → 실행


## Comment

당연한 말이지만 상용 서비스가 성능 더 좋다. (로컬 모델은 제일 성능 좋은 것도 정확도 70이 안 된다)

하책으로 중간에 검증하는 단계를 붙여놔서 검증만 잘하면 딱히 문제는 없겠지만 편의성도 떨궈졌다.

그렇다고 그 누구도 관리 중인 DB를 상용 서비스에 다 올리는 짓은 못(안) 할테니 아쉬운대로 이런 게 나왔다.
